Collaboration for Cloud Teams

Facilitate collaboration between Application Developers, Cloud operations Team, NOC Teams, DBAs, Information security teams, Release Teams for your software product delivery.

  • Facets.cloud Stack Definition Language (FSDL)
  • Git Based Stack definition and workflow
  • Standard and Custom validations options
Continuous Delivery of Software Product

The FSDL based stack definition can be used to create a completely working instance of your software product on any cloud through Facets.cloud Control Plane that is private to your deployment. Additionally, the control plane enables continuous mutations of the software product.

  • Spin-off and teardown deployments/clusters in minutes
  • Automated scheduled delivery of the entire software product
  • Trigger notifications to QA suites, ChatOps tools and certify the release automatically/manually
Comprehensive Day 2 Operations

Facets.cloud Run-time provides a comprehensive suite of tools and processes to simplify Day 2+ operations. Agents are deployed to reduce the work required for the management of the underlying cloud and Kubernetes infrastructure of your software product. Best Practice Policies and guardrails are enforced in a consistent manner across all deployments.

  • Pre-configured Alerts and Dashboards for Observability and Compliance
  • Cloud Agents deployed for various purposes like log collection, spot management
  • Extend easily while maintaining the standards using Facets interfaces

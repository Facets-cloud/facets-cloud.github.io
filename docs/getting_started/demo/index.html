    Scenario

    Awesomeness Inc. is a company having two product suites “S3 file uploader” and “Todo App”. S3 uploader application needs to be deployed in a VPC and has various components to it. Todo app too needs a separate VPC and has couple of microservices and database in it.

    Awesomeness Inc. is a Facets customer and has its own control plane. We will sit through the company's journey of creation of these stacks in Facets product.
